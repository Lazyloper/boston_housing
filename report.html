6.8

  • Separate data into k buckets and train k times, each times using a different bucket as validating set and the rest as training set. Finally take the average.
  • GridSearchCV will use cross validation for each possibility of parameters.
  • cvresults tells us the result of all different parameters or combinations.
  • There's risk of overfitting. Cross validation use different training set to avoid our model from memorizing data. In addtion, cross validation enables us to use all the data expect for testing data to train. Without cross validation, some useful data with good features will not be used for training.
